# Inventory Stock Management System

A simple and efficient **Inventory Stock Management System** designed to help track products, manage stock levels, monitor sales, and generate useful insights for business operations.

---

## 🚀 Features

* **Add, update, and delete products**
* **Track stock levels** with automatic adjustments on sales or restocks
* **Product List**
* **Sales management**
* **Sales Dashboard** 
* **Secure authentication** 
* **Responsive UI** 

---

## 📦 Tech Stack
- **React** — Frontend library
- **Spring Boot** — Backend
- **Spring Security** — Secure authentication
- **CSS** — Styling
- **MySQL** — Database


---

## 🛠️ Installation

### Prerequisites

Make sure you have the following installed:

* Node.js (if React/Node project)
* Java 8/11+ (if Java backend)
* MySQL/PostgreSQL/MongoDB (choose yours)

### Steps

1. Clone the repository:

```
git clone  [https://github.com/danisasubhani996/InventoryManagement.git]
```

2. Navigate to the project directory:

```
cd InventoryManagement
```

3. Install dependencies:

```
npm install
```

4. Start the server/UI:

```
npm start
```
